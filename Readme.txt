Это приложение демонстрирует работу сглаживания изображений 
с помощью линейной фильтрации. Коротко - матрица(маска) разных размеров
пробегает по изображению и с помощью не сложных вычислений меняет 
значения каждого пикселя. Для создания маски использовалась библиотека
EmguCV, которая является оболочкой OpenCV. 
Создание графического интерфейса - WPF.
